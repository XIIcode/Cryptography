This is a python script that Encrypts and Decrypts data based on Caesar Cipher.
N.B USE THE SAME KEY-BIT(MIN 0, MAX 25)TO ENCRYPT AND DECRYPT
N.B WHEN USING HARDENED CAESAR ENSURE TO MATCH THE ORDER OF ENCRYPTION KEYS WELL WHEN DECRYPTING
BRUTEFORCE USED BASED ON THE VERSIONS OF CYPHERS ONLY.
To run
clone the repository
cd Caesar_Cryptography
check the one you wish to run 
E.G python3 Caesar_1.0.0.1.py
